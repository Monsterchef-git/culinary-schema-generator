# 👨🍳 Culinary Schema Generator v2.0

> **"En el sector gastronómico, no basta con cocinar bien; hay que ser encontrable."**

Este script interactivo en Node.js automatiza la creación de **Datos Estructurados (JSON-LD)** bajo el estándar de [Schema.org](https://schema.org/Recipe). Su objetivo es permitir que Google indexe recetas con **Rich Snippets** (estrellas, tiempos de cocción y autoría), aumentando drásticamente el CTR (Click-Through Rate) de sitios web de restaurantes y portfolios de chefs.

---

## 🚀 ¿Por qué es importante?

Sin metadatos, Google ve tu receta como simple texto. Con este generador, transformas tu contenido en una **entidad semántica**.

- **SEO Local:** Vincula tu nombre y ubicación (Medellín/España) con platos específicos.
- **Rich Results:** Aparece en las "Tarjetas de Recetas" de Google con fotos y tiempos.
- **E-E-A-T:** Establece tu autoridad como Chef Profesional ante el algoritmo.

---

## 🛠 Instalación y Uso

Este es un script de terminal ligero, sin dependencias externas.

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/Monsterchef-git/culinary-schema-generator.git
   cd culinary-schema-generator
   ```

2. **Ejecuta el generador:**
   No requiere `npm install` porque solo utiliza módulos nativos de Node.js.
   ```bash
   node index.js
   ```

3. **Interactúa:**
   Responde a las preguntas en la terminal (Nombre del plato, Chef, Ingredientes, etc.) y obtendrás instantáneamente tu JSON listo para copiar y pegar en tu sitio web.
